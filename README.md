# CH 2. 사칙연산 계산기 과제

> 과제 목표: 자바 문법의 실습과 객체 지향 개념의 적용

### 수행한 단계


`필수` [STEP 1] : 클래스 없이 자바의 기본 문법만을 사용하여 구현한 계산기

`필수` [STEP 2] : 객체 지향 개념을 적용하여 클래스 정의, 연산을 수행하는 메소드 작성한 계산기

`도전` [STEP 3] : Java의 Enum, 제네릭, 람다 & 스트림을 적용한 계산기

___

### 프로세스 흐름

1. **첫 번째 숫자 입력**
2. **두 번째 숫자 입력**
3. **연산자 선택**
4. **제네릭 기반 계산 로직 실행 및 결과 출력**
5. **연산 결과를 내부 컬렉션(List)에 저장**
6. **가장 먼저 저장된 이력 삭제 여부 확인 및 처리**
7. **사용자가 입력한 기준값보다 큰 결과만 Stream으로 필터링하여 출력**
8. **계산 반복 여부 확인**

___
### 📂 클래스 구조

```
CalculatorProject

        └─ Main.java                   # 전체 흐름 통합 실행
```
